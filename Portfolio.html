Sobre mí

PEDRO MULA MONTESINOS

Soy un apasionado programador web en busca de oportunidades para aprender y crecer en la industria. Aunque no tengo experiencia profesional, estoy lleno de entusiasmo y compromiso para enfrentar nuevos desafíos.
Mi enfoque principal es la creación de sitios web modernos y funcionales, utilizando tecnologías como HTML, CSS y JavaScript. Me esfuerzo por seguir las mejores prácticas de desarrollo web y mantenerme actualizado con las últimas tendencias y herramientas del campo.
En mi portfolio, encontrarás proyectos personales que reflejan mi dedicación para adquirir nuevas habilidades y aplicarlas en entornos prácticos. Cada proyecto es una oportunidad para aprender y mejorar, y estoy emocionado de compartir mi progreso contigo.
Mis habilidades incluyen la creación de diseños responsivos, la implementación de interactividad mediante JavaScript y la integración de APIs para mejorar la funcionalidad. Además, tengo experiencia con herramientas como Git para el control de versiones y he trabajado en entornos de desarrollo colaborativos.
Estoy buscando activamente oportunidades para colaborar en proyectos, contribuir al éxito de equipos y seguir desarrollándome como profesional. Mi enfoque es aprender de manera continua y aplicar mis conocimientos de manera creativa para superar cualquier desafío que se presente.
Gracias por visitar mi portfolio.
